
# what's GTD?
GTD(get thing done),a template handle thing, including the following 4 process
1. capture:record user's requirement , thoughts to be implemented collect them into inbox.
2. clarify:classify all the thing in inbox to different list.
	- quick action list: could be done in 2min.
	- complex list: need to be split to more action
	- time related list:have a due date need a reminder
	- waiting for list:wait for somebody or some other resource
	- other more tag each list may have a different hook action for example the time related list may send a notification at due date.

3. organize:make a implementation plan(including due date)
4. reflect:review and adjust the plan according to the res of implementation.
5. engage:implement the task according to the plan.
