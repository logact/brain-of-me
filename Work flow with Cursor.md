# Initiation
## 1. initiate the tech context
`context/tech-spec.md`: what tech we use
## 2. initiate the product overview 
`context/product-overview.md`: the product background the agent need to know.
## 3. break down the PRD to the features
`docs/prd.md`: user product spec that describe the what's the product like.
`docs/features.md`: the features that is break down from the `prd.md` and is provided to daily execution.

## 2.working with cursor to implement it
## 3. maintain the state of the project

`context/cur-feature.md` current working-on feature
`context/cur-status.md` current project status
`docs/process.md` features process and schedule.
